# Описание экранов (UI)

## 1. AuthScreen (Общий)
* В выпадающем списке "Оператор" всегда первой строкой идет **Admin**.
* После успешного входа под Admin происходит редирект на `AdminDashboard`.

## 2. Dashboard
* Список активных задач и кнопка "Новый осмотр" (иконка камеры).

## 3. OCRScanScreen
* Рамка-видоискатель для сканирования номера контейнера.
* Валидация в реальном времени.

## 4. InspectionScreen
* Динамический заголовок с шагом (1/7) и подсказкой из DOMAIN_LOGIC.md.
* Превью сделанного снимка.

## 5. SummaryScreen
* Сетка 2x4 с превью всех 7 фотографий.
* Кнопка "Сформировать PDF и отправить".

## 6. AdminDashboard (Экран Администратора)
* **Навигация**: Нижний бар (BottomNavigationBar) с двумя иконками:
    1. **"Операторы"**: 
       - Список текущих операторов (Оксана, Андрей и др.).
       - Кнопка "Добавить" (имя + PIN).
       - Иконка "Корзина" у каждого имени для удаления.
       - Иконка "Скачать" для сохранения списка в память телефона.
    2. **"Настройки"**:
       - Поля ввода: URL Nextcloud, Логин, Пароль/Токен.
       - Кнопка "СОХРАНИТЬ".
       - **Индикатор подключения**: После нажатия "Сохранить" выполняется тестовый запрос к WebDAV. При успехе появляется зеленый индикатор (точка или галочка) "Подключено".
