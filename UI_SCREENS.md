# Описание экранов (UI)

## 1. AuthScreen
* Выбор оператора (Оксана/Андрей) и смены (1-4).
* Ввод PIN-кода.

## 2. Dashboard
* Список активных задач и кнопка "Новый осмотр" (иконка камеры).

## 3. OCRScanScreen
* Рамка-видоискатель для сканирования номера контейнера.
* Валидация в реальном времени.

## 4. InspectionScreen
* Динамический заголовок с шагом (1/7) и подсказкой из DOMAIN_LOGIC.md.
* Превью сделанного снимка.

## 5. SummaryScreen
* Сетка 2x4 с превью всех 7 фотографий.
* Кнопка "Сформировать PDF и отправить".
