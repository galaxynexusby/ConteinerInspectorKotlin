# Техническая спецификация (Kotlin)

## Модули
* **OCR**: ML Kit Text Recognition (Bundled).
* **Камера**: CameraX API.
* **База данных**: Room (очередь осмотров).
* **Синхронизация**: WorkManager + OkHttp (WebDAV).
* **Отчеты**: `android.graphics.pdf.PdfDocument`.

## Логика папок (Nextcloud)
Файлы должны загружаться по пути:
`/[Год]/[Месяц]/Смена[N]/[Номер_Контейнера]/`
* **Год/Месяц**: Определяются автоматически на русском языке.
* **Смена**: Выбирается оператором при входе (1-4).
