# Правила для ИИ

* Весь текст в интерфейсе — на **русском языке**.
* Использовать **Material3** и крупные кнопки для работы в перчатках.
* Названия месяцев в путях Nextcloud писать кириллицей (Январь, Февраль и т.д.).
* Реализовать `NavGraph` для связки всех 5 экранов.

* Всегда отключай датчики (сенсоры) и камеру при переходе приложения в background для оптимизации батареи.

# Правила интерфейса

## Тема и Цвета
* **StatusBar**: Должен быть прозрачным или иметь цвет `Surface` (задний фон приложения), чтобы не было серой полосы сверху. Используй `enableEdgeToEdge()` в Activity.
* **Навигация**: Для админа и оператора использовать разные Navigation Graph.
